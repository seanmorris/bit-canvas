# BitCavas

https://bit-canvas.seanmorr.is/

## About

Extract & decompress pixel & text from ROM files of classic games.

* Gameboy Encodings:
	* 2bpp 8x8 tiles
	* 1bpp 8x8 tiles
	* 1bpp 2px columns
	* 4-bit RLE/Delta compressed sprites (pokemon algorithm)
	* Generation 1 Pokedex data format

* Misc:
	* 8 bit greyscale
	* 1 bit b & w
